MICROSOFT ACCESS SUPPORT

How to organize the Access objects by type, so that all the tables are grouped together, all the queries are grouped together, etc?
* Click the pull-down menu at the top of the navigation pane.
Choose Object Type.
* The objects will now be grouped according to object type. To view the objects within each type, click on the object name one time to expand the list.
What is the file extension for an Access 2013 database?
The default file extension for an Access 2013 database is .accdb. 
How do I extract some of the data in my database and use it in Excel?
Open Excel and create a new blank worksheet (File | New).
Open the Access database.
Resize or rearrange the Excel and Access windows so that you can see both at the same time.
Select the object you wish to export (either a table or a query).
Left click on the object, and drag it to the Excel worksheet.
Release the mouse button while pointing at cell A1.
Dont forget to save the Excel worksheet.
Note: Keep in mind that the data in the worksheet will not be updated if the Access data is updated.

How do I create a new Access database?
From the Start Menu, choose All Programs, then Microsoft Office, then Microsoft Office Access 2010. This will open Access with the New command on the File tab selected.
Click the Blank Database icon.
Under filename, change the default filename , Database1  to the filename that you would like to use for your database. Do not modify the file extension.
Click on the folder icon to the right of the filename if you would like to change the save location to a different folder. Browse to the desired folder and click OK. 
Click Create.
